Question-3: What is event delegation in the DOM?

When we add a event listener on the parent ,we can get easily access of the full parent such as parent's included everyNode.for this reason we cas easily handle the child elements easily.here it works by using event bubbling system.

Question-4: What is event bubbling in the DOM?

When we add a event listener function on a target element like click function on the target element .it goes propagates from the target element to its parent to grandparent elements in the DOM tree.at last it goes to root of the document's root element and then the output are shown.
